=====================
Install and configure
=====================

This section describes how to install and configure Masakari-monitors services
on the compute node.

This section assumes that you already have a working OpenStack environment
with at least the following components installed: Nova, Glance, Cinder,
Neutron and Identity.

Note that installation and configuration varies by distribution.
